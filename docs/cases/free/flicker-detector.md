# 频闪检测仪

## 项目简介
用于检测 LED 灯具与显示设备的频闪表现，适合做照明与视觉舒适度的基础评估。

## 功能特点
- 快速采集光强变化数据
- 支持基础频闪趋势观察
- 便携结构，适合现场测试
- 可作为后续算法验证平台

## 未来规划
- 增加亮度、色调的测量，预计使用TCS34725实现

## 硬件组成（简要）
- 光源检测放大滤波
- 充电模块
- 显示模块
