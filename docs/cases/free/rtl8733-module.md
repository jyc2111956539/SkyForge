# RTL8733 蓝牙 Wi-Fi 二合一 USB 模块

## 项目简介

RTL8733 模块是一个面向嵌入式平台的蓝牙 + Wi-Fi 二合一无线扩展模块。项目核心目标是通过 USB 接口补齐主控平台的无线连接能力，避免占用额外 SDIO 资源。

在一些接口资源紧张的平台上，例如 RK3506 这类没有多余 SDIO 接口的主控，如果仍然需要同时接入蓝牙和 Wi-Fi，使用 USB 接口的 RTL8733 模块会更容易集成，也更方便后续适配驱动和调试。

## 项目定位

这个模块主要用于给 Linux 开发板、工业控制板、网关设备或便携终端增加无线连接能力。相比传统 SDIO Wi-Fi 模块，它的优势是接入方式更直接：主控侧只需要提供 USB 接口，即可扩展蓝牙和 Wi-Fi。

适合以下情况：

1. 主控没有空余 SDIO 接口。
2. SDIO 已经被存储、显示或其它高速外设占用。
3. 项目需要同时支持蓝牙和 Wi-Fi。
4. 希望无线模块接口更通用，方便跨平台复用。

## 功能特性

### 蓝牙能力

1. 支持蓝牙 4.0 低功耗。
2. 支持蓝牙 5.2 双模。
3. 支持 AP 模式相关应用场景。
4. 可用于低功耗外设连接、近距离数据交互、配网和设备发现等场景。

### Wi-Fi 能力

1. 支持 Wi-Fi 无线联网。
2. 支持 WPA / WPA-PSK 安全机制。
3. 支持 WPA2 / WPA2-PSK 安全机制。
4. 可用于设备联网、局域网通信、远程配置和数据上传。

### 接口能力

1. 主控侧通过 USB 接入。
2. 不额外占用 SDIO 接口。
3. 适合 SDIO 资源较少或已被占用的平台。
4. 便于在不同主控平台之间复用硬件方案。

## 典型应用场景

| 场景 | 说明 |
|---|---|
| RK3506 平台无线扩展 | RK3506 无多余 SDIO 接口时，可通过 USB 增加蓝牙和 Wi-Fi |
| 工业网关 | 增加无线联网、蓝牙配置或近场维护能力 |
| Linux 开发板 | 用作通用无线扩展模块，降低接口占用压力 |
| 便携设备 | 通过单个模块同时提供 Wi-Fi 与蓝牙能力 |
| 调试与样机验证 | 快速验证无线功能，不需要重新规划 SDIO 资源 |

## 系统连接方式

典型连接关系如下：

```text
主控平台 USB Host -> RTL8733 USB 模块 -> 蓝牙 / Wi-Fi 无线连接
```

如果主控平台的 SDIO 已经分配给其它外设，RTL8733 模块可以作为 USB 无线扩展方案，减少硬件资源冲突。

## 设计优势

1. **接口占用少**：通过 USB 接入，不占用主控 SDIO。
2. **功能集成度高**：一个模块同时提供蓝牙和 Wi-Fi。
3. **平台适配灵活**：适合 RK3506 等接口资源有限的平台。
4. **安全机制完整**：Wi-Fi 支持 WPA、WPA-PSK、WPA2、WPA2-PSK。
5. **后续扩展方便**：可以继续补充驱动适配、天线测试和吞吐实测内容。
