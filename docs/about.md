# 关于凌云工坊

凌云工坊（SkyForge）是一个面向嵌入式开发者与电子设计爱好者的技术分享站点。
我们聚焦“学得会、下得到、做得出”三件事：用清晰教程帮助学习，用结构化资料支持复用，用可落地项目连接实践。

## 我们在做什么

1. 教程体系  
覆盖 STM32、ESP32、Arduino、ARM 与电路设计等方向，强调可复现的实战过程。

2. 案例实战  
展示电子与软件项目案例，提供方案简介、设计亮点和后续可扩展方向。

3. 资料库  
提供 PDF / DOCX / ZIP 等下载型资料，方便快速查阅与工程复用。

4. 周边商城  
围绕小规模项目配套的板卡、工具与开发周边，服务学习与打样需求。

## 我们的内容标准

1. 可复现  
教程与方案尽量提供完整步骤，减少“只看懂、做不出”的情况。

2. 可检索  
内容按主题分类，便于后续长期积累和快速定位。

3. 可复用  
优先沉淀模板化资料，帮助你在新项目中直接套用。

4. 可审核  
评论与互动内容采用先审核后展示机制，保持讨论质量。
