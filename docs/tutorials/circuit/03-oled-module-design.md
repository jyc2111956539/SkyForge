# 0.96/1.3 寸 OLED 模块设计详解（I2C/SPI 四版本）

## 项目说明
本文面向 I2C/SPI 双接口的 OLED 模块设计，覆盖 0.96 寸与 1.3 寸两种尺寸。

## 硬件说明
- 稳压芯片采用 `SC662K-3.3V`
- 目前测试可用 `SSD1306` 与 `SH1106` 两类屏控
- 同时提供 I2C 与 SPI 两套接口版本

## 版本划分
- I2C 0.96 寸
- I2C 1.3 寸
- SPI 0.96 寸
- SPI 1.3 寸
