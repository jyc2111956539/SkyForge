# ESP32 MQTT 上云实践

## 你将学到什么
- MQTT 连接与订阅发布
- 断线重连与状态同步

## 硬件准备
- ESP32 开发板
- MQTT Broker

## 步骤
1. 初始化客户端并连接 Broker。
2. 订阅控制主题。
3. 发布传感器数据并验证。

## 常见问题
- TLS 证书配置错误
- 心跳超时断连
