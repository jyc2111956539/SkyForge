# ESP32 Wi-Fi 连接与状态监测

## 你将学到什么
- Wi-Fi 连接流程
- 连接失败重试策略

## 硬件准备
- ESP32 开发板
- 可用 Wi-Fi 网络

## 步骤
1. 配置 SSID 与密码。
2. 启动连接并读取状态。
3. 打印网络参数用于验证。

## 常见问题
- 2.4G/5G 网络混淆
- 密码编码错误
