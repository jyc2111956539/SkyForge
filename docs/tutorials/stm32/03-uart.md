# STM32 UART 串口通信

## 目标
- 完成串口初始化
- 打通 PC 与开发板的数据收发
- 建立可复用的日志输出接口

## 准备
- STM32 开发板
- USB 转串口模块或板载串口

## 步骤
1. 配置波特率、数据位、停止位。
2. 实现发送函数并打印测试字符串。
3. 增加接收中断，回显收到的数据。

## 常见问题
- 乱码通常由波特率不一致导致。
- 收不到数据先检查 TX/RX 是否交叉连接。
