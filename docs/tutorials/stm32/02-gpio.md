# STM32 GPIO 点灯实战

## 你将学到什么
- GPIO 输入输出模式
- 点灯最小闭环
- 简单延时与状态切换

## 硬件准备
- STM32 开发板
- LED + 电阻

## 步骤
1. 配置 GPIO 为推挽输出。
2. 在循环中翻转引脚电平。
3. 观察 LED 亮灭并调整延时。

## 常见问题
- LED 不亮先检查极性和限流电阻。
- 若引脚无变化，确认时钟和端口初始化。
