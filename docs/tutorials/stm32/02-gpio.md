# STM32 GPIO 点灯实战

## 你将学到什么
- GPIO 工作模式与输出控制
- 点灯程序的最小闭环

## 硬件准备
- STM32 开发板
- LED + 电阻

## 步骤
1. 配置 GPIO 口模式。
2. 编写点灯逻辑并下载运行。
3. 用示波器或万用表确认引脚状态。

## 常见问题
- 引脚复用冲突
- 灯不亮但程序正常运行
