# STM32 入门与环境搭建

## 你将学到什么
- STM32 学习路径和常见开发流程
- 基础工程结构与调试方式
- 后续 GPIO/UART/PWM 学习怎么衔接

## 准备内容
- STM32 开发板一块
- USB 数据线
- 串口工具与 IDE

## 建议步骤
1. 安装工具链并确认串口驱动正常。
2. 新建一个最小工程，编译并下载到板子。
3. 通过串口输出验证程序已运行。

## 常见问题
- 下载失败时先检查供电和下载器连接。
- 串口无输出时先确认波特率和引脚配置。
