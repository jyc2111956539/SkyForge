# STM32 入门：开发环境与最小工程

## 你将学到什么
- 如何搭建 STM32 开发环境
- 如何创建并烧录最小工程
- 如何定位最常见的启动失败问题

## 硬件准备
- STM32 开发板
- USB 数据线

## 步骤
1. 安装工具链并验证编译器。
2. 创建最小工程并配置时钟。
3. 下载到开发板并观察串口输出。

## 常见问题
- 无法识别串口
- 烧录成功但程序不运行
