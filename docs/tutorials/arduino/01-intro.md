# Arduino 入门：第一个传感器项目

## 你将学到什么
- Arduino IDE 基础使用
- 传感器读取与串口输出
- 从实验到小项目的组织方式

## 硬件准备
- Arduino UNO/Nano
- 基础传感器模块

## 步骤
1. 连接传感器并确认引脚。
2. 编写读取代码并上传。
3. 校准阈值并输出结果。

## 常见问题
- 串口乱码
- 读取数据抖动
