# Arduino 传感器采集与串口可视化

## 你将学到什么
- 模拟量采样
- 串口绘图观察趋势

## 硬件准备
- Arduino UNO/Nano
- 模拟传感器模块

## 步骤
1. 读取传感器数据。
2. 串口输出格式化。
3. 使用串口绘图工具观察曲线。

## 常见问题
- 采样噪声过大
- 串口刷新率过高导致卡顿
