# Arduino OLED 显示实战

## 你将学到什么
- I2C 屏幕驱动基础
- 页面刷新与布局优化

## 硬件准备
- Arduino UNO/Nano
- 0.96 寸 I2C OLED

## 步骤
1. 引入显示库并初始化。
2. 绘制文本与图形。
3. 结合传感器数据显示。

## 常见问题
- I2C 地址不匹配
- 刷新闪烁
